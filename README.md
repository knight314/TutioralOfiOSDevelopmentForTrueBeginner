# TutioralOfiOSDevelopmentForTrueBeginner
零基础学习iOS(Objective C)的教程

本教程的目的是讲解尽量少的内容和知识就可以让零基础的朋友开始编写简单应用。采用纯代码,布局以frame为主。
在学习过程中很有可能遇到不容易理解的新概念，请以能够独立完成课后的作业为评价依据。如果可以完成，请学习下一课，如果不能继续完成，请继续练习本课内容。
